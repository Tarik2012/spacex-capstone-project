Executive Summary

This project is part of the IBM Data Science course and simulates a real-world scenario: working as a data scientist in a company called SpaceY, which aims to compete with SpaceX.

The main objective is to analyze public launch data from SpaceX, extracted from its official REST API and other sources, to predict whether the first stage of the Falcon 9 rocket will be reused in a given launch.

This will allow SpaceY to estimate launch costs and make strategic decisions to reduce expenses and improve competitiveness in the aerospace market.

Project stages:

  • Data collection from APIs and Wikipedia
  • Dataset cleaning and normalization
  • Exploratory data analysis (EDA)
  • Visualization with interactive graphs
  • Predictive classification model
  • Conclusions and professional presentation in Django
