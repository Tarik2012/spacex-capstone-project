Data Collection

In this section, we outline the methods used to collect raw data about Falcon 9 rocket launches. Two main sources were used:

  • SpaceX REST API: Provides structured JSON data on past launches, including payloads, rocket IDs, landing outcomes, and more.
  • Wikipedia: Specific tables were scraped using BeautifulSoup to enrich the dataset with additional contextual information such as mission names and outcomes.

The collected data includes technical and mission-specific information which was later cleaned and normalized to build a high-quality dataset ready for analysis and modeling.

Tools used:

  • requests – To access and retrieve data from the SpaceX API
  • pandas – For data wrangling and normalization
  • BeautifulSoup – For scraping HTML tables from Wikipedia
  • json_normalize – To flatten nested JSON structures into tabular format
